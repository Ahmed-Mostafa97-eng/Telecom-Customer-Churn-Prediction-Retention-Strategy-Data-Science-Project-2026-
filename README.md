# Telecom-Customer-Churn-Prediction-Retention-Strategy-Data-Science-Project-2026-
 Built an end-to-end ML pipeline on 100,000 customer records (XGBoost, ROC-AUC 0.69) and translated it into a targeted retention plan projected to deliver ~$1.17M net benefit/year (467% ROI). Python, scikit-learn, XGBoost, pandas.
